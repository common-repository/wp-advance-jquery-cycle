== Description ==

A very neat and trendy looking slideshow which allows you to add images to your wordpress powered website and rotate them as you like. You can apply settings according to your preferences and liking. This plugin supports wordpress version 3.0 to the latest one available. It supports all major browsers.

NOTE: For the current version user must have a working knowledge of HTML & CSS. We will eliminate this in the new release.


[For styling help of this plugin please click on this link](http://www.base29.com/wp-advance-jquery-cycle-styling-tutorial/)

= Key Features =

*	Supports all major formats png, gif, jpg ETC.
*   Customisable design 
*   Works in all major browsers - IE7, IE8, IE9, Safari, Firefox, Chrome
*   Works on mobile devices including iPhone/iPad
*	Add the slideshow to any post/page using shortcode `[wp_jcycle]`

== Installation ==


1. Upload plugin folder `wp-advance-jquery-cycle` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
